API Flow

  • Get the input URL (or URLs)
  • Create your JSON body
  • Specify your output URL (or URLs)
  • Hit the endpoint
  • Poll for response
  • Repeat. Profit. Etc.
Example 1: Remove Background 

{
  "input": {
    "href": "string",
    "storage": "external"
  },
  "options": {
    "optimize": "performance",
  },
  "output": {
    "href": "string",
    "storage": "external",
    "overwrite": true,
    "color": {
      "space": "rgb"
    },
    "mask": {
      "format": "soft"
    }
  }
}
